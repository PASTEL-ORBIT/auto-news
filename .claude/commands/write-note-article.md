# Note.com 記事作成

Note.com に投稿する記事を作成する。以下のガイドラインに従うこと。

## 文章構成

- **逆三角形**: 冒頭に一番伝えたいメッセージ → 説明 → 補足の順で書く
- 2〜3文ごとに改行を挟む（スマホ閲覧を意識）
- 長い記事には目次をつける
- 図版・写真・見出し・箇条書きを一定間隔で挟み、リズム感を出す

## 文体

- 1文を短くする（不要な言葉は削る）
- 修飾語と被修飾語を近くに置く
- 漢字・ひらがな・カタカナのバランスを調整し、漢字が連続しすぎないようにする
- 親しみやすい口調を心がける
- **本文（地の文）**: ですます調＋ややくだけた感じ（「～が良いでしょう」など）
- **箇条書き・列挙**: 体言止め風にする（「～できるように」「～可能」「～強い」など。ですます調にしない）
- 表（テーブル）は使わず、テキストベースで説明する
- 引用するプロンプト例などの英文は日本語に訳して掲載する

## 公開前チェック

- 声に出して読んでつっかえる箇所がないか確認する
- タイトルと見出し画像で内容が想像できるようにする

## 出力先

記事ファイルは `Note/` フォルダに作成する。
