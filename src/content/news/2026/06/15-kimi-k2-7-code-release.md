中国のMoonshot AIは、Mixture-of-Expertsアーキテクチャを採用した1兆パラメータ（活性化パラメータ320億）のコーディング特化モデル「Kimi K2.7 Code」を公開した。

## 詳細

K2.7 Codeは、APIおよびHugging Face上でModified MITライセンスのもと公開されており、前モデル「K2.6」と比較して推論に使用するトークン数を約30%削減しながら、Kimi Code Bench v2で21.8%、Program Benchで11.0%、MLS Bench Liteで31.5%の性能向上を達成したという。価格は入力100万トークンあたり0.95ドル、出力100万トークンあたり4.00ドルに設定されている。

## 影響/意義

GPT-5.5やClaudeなど主要モデルと比較して最大12倍安価な価格設定により、長時間にわたるコード生成・デバッグを行うAIエージェント向けの選択肢として、開発コストを重視する企業からの注目が集まると見られる。
