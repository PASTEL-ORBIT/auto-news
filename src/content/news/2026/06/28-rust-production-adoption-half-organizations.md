JetBrains が2026年2月に公開した「State of Rust Ecosystem 2025」によると、調査対象企業の約半数が Rust を本番環境で実質的に利用していることが明らかになった。2023年時点での採用率は38〜39%程度であり、2年間で大幅な拡大が確認された。

Rust は Stack Overflow Developer Survey において9年連続で「最も愛されているプログラミング言語」の首位を維持し、2026年時点での支持率は83%に達する。過去12ヶ月間の利用者数は2,200万人超で、商業利用は2021〜2024年の3年間で68.75%増加した。Google、Microsoft、Amazon、Meta、Cloudflare などの大手テック企業がシステムレベルの実装に Rust を採用しており、Microsoft は Windows のロー・レベルコンポーネントの一部を Rust で書き直す取り組みを進めている。

AI ツールによるコード生成が普及した現在でも、メモリ安全性とネイティブパフォーマンスを両立する Rust の価値は高まり続けている。パッケージマネージャ「Cargo」もクラウド・インフラツールの中で最高評価を獲得しており、エコシステムの成熟が実用化の加速を後押ししている。
