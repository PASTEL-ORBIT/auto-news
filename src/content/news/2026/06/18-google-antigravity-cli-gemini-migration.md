Googleは2026年6月18日をもってGemini CLIのサービスを正式終了し、開発者向け統合プラットフォーム「Antigravity CLI」への完全移行を実施した。個人・無料利用ユーザーは本日以降、旧ツールへのアクセスが不能となる。

GoogleはGoogle I/O 2026（5月19日）にて、開発者ツールを「Antigravity」ブランドへ統合する方針を発表。Gemini CLIと合わせてGemini Code Assist IDEエクステンションも廃止し、新プラットフォームへの一本化を進めてきた。

Antigravity CLIはGo言語実装の高性能ターミナルインターフェースで、統合エージェントハーネスによるマルチエージェントオーケストレーションに対応する。同社のデスクトップアプリケーション「Antigravity 2.0」とも連携し、複雑なワークフロー自動化を実現する設計となっている。

ただし、エンタープライズ向けGemini Code Assistのライセンス購入済み組織は引き続き旧CLIを使用できるため、法人ユーザーへの影響は限定的だ。

一方、旧Gemini CLIがApache 2.0ライセンスのオープンソースプロジェクトとして6,000件超のコミュニティプルリクエストを受け入れてきた経緯があるのに対し、Antigravity CLIは非オープンソースとして提供される。この方針転換に対して、オープンソースコミュニティから強い批判が集まっている。
